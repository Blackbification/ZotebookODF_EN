## Cómo crear el GIF demo

1. Usa [ScreenToGif](https://www.screentogif.com/) (Windows) o [Kap](https://getkap.co/) (Mac)
2. Graba:
   - Hacer clic en citas en NotebookLM
   - Ver el badge actualizarse
   - Abrir el popup y exportar
3. Optimiza el GIF (máx 5MB para GitHub)
4. Guárdalo como `demo.gif` en esta carpeta
